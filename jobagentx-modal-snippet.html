JobAgentX early-access modal

Open this file in a browser to preview the modal (it pops after ~5s). To deploy: copy everything between the COPY FROM HERE / TO HERE markers in this file's source and paste it right before </body> on your JobFairX marketing pages and inside the app.

  1. Create a HubSpot form with fields Email, Company, Message.
  2. In the snippet, set HS_PORTAL_ID and HS_FORM_GUID (HubSpot → Marketing → Forms → Share/Embed).
  3. Point LEARN_MORE_URL at your live early-access page.
  4. Paste the block before </body>. It shows once per visitor (remembers via localStorage), is dismissible, and won't re-nag.
Tweak SHOW_AFTER_MS (default 5000) for timing. It will not appear during the same session after a visitor dismisses or submits, so it won't interrupt your core JobFairX flow repeatedly.
