  1. Create a HubSpot form with fields Email and First name.
  2. In the snippet, set HS_PORTAL_ID and HS_FORM_GUID (HubSpot → Marketing → Forms → Share/Embed).
  3. Paste the block before </body>. It shows once per visitor (remembers via localStorage), is dismissible, and won't re-nag.
Tweak SHOW_AFTER_MS (default 5000) for timing. It will not reappear in the same session after a visitor dismisses or submits, so it won't interrupt your core JobFairX flow.
